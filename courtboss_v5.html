Google Sheets Setup
Every match auto-logs to your spreadsheet
01
Create Your Google Sheet

1. Go to sheets.new to open a blank spreadsheet.
2. Rename it CourtBoss Results (or anything you like).
3. In Row 1, type these headers across columns A–L:
A: Timestamp
B: Sport
C: Session
D: Round
E: Court
F: Team A P1
G: Team A P2
H: Team B P1
I: Team B P2
J: Score A
K: Score B
L: Winner

02
Add the Apps Script

1. In your Google Sheet, click Extensions → Apps Script
2. Delete ALL existing code in the editor
3. Copy and paste the script below, then click 💾 Save (Ctrl+S)

03
Deploy as a Web App

1. In Apps Script, click Deploy → New Deployment
2. Click the gear icon ⚙ and select Web App
3. Set Execute as: Me
4. Set Who has access: Anyone
5. Click Deploy
6. Authorise when prompted (click Allow)
7. Copy the Web App URL — it looks like:
https://script.google.com/macros/s/ABC…/exec

04
Paste Your URL & Connect

Paste the Web App URL below and click Save. CourtBoss will sync every match result automatically.

Deployment Tips

• If you update the script, you must create a new deployment — editing an existing one won't update the live URL.
• If you see an "Authorisation required" error, re-deploy and click Allow when Google asks for permission.
• Results sync automatically each time you click Confirm & Next Round.
• Both Badminton and Pickleball results go to the same sheet, labelled by sport.
